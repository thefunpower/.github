## 安装
~~~
composer thefunpower/installer 
~~~

安装`installer`
真实包含以下composer

~~~
thefunpower/core 
thefunpower/vue 
thefunpower/db_medoo 
thefunpower/license_checker 
thefunpower/auth 
~~~

其中除了`thefunpower/db_medoo`为MIT协议，其他包均为商用须要授权协议。
